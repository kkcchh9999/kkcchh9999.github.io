2007년

문제

2007년, 1924번

오늘은 2007년 1월 1일 월요일이다. 그렇다면 2007년 x월 y일은 무슨 요일일까? 이를 알아내는 프로그램을 작성하시오.

입력 : 첫째 줄에 빈 칸을 사이에 두고 x(1 ≤ x ≤ 12)와 y(1 ≤ y ≤ 31)이 주어진다. 참고로 2007년에는 1, 3, 5, 7, 8, 10, 12월은 31일까지, 4, 6, 9, 11월은 30일까지, 2월은 28일까지 있다.

출력 : 첫째 줄에 x월 y일이 무슨 요일인지에 따라 SUN, MON, TUE, WED, THU, FRI, SAT중 하나를 출력한다.

풀이 과정

일주일은 7일이고, 이를 이용해 나머지 연산을 통하여 문제를 해결한다. x월일 경우, (1월의 일 수(31) + 2월의 일 수(28)+….(x-1)월의 일 수(??) + y )을 하면 총 몇일인지 변환이 가능하고, 여기에 % 7 연산을 한 결과에 따라 요일을 출력하는 방식으로 해결한다.

코드

#include

using namespace std;

int main() {

int x, y, tmp = 0;
cin >> x >> y;

for (int i = 1; i < x; i++)
{
    if (i == 1 || i == 3 || i == 5 || i == 7 || i == 8 || i == 10 || i == 12)
        tmp = tmp + 31;
    else if (i == 2)
        tmp = tmp + 28;
    else
        tmp = tmp + 30;
}
tmp = tmp + y;
if (tmp % 7 == 1)
    cout << "MON" << endl;
else if (tmp % 7 == 2)
    cout << "TUE" << endl;
else if (tmp % 7 == 3)
    cout << "WED" << endl;
else if (tmp % 7 == 4)
    cout << "THR" << endl;
else if (tmp % 7 == 5)
    cout << "FRI" << endl;
else if (tmp % 7 == 6)
    cout << "SAT" << endl;
else
    cout << "SON" << endl;

}

```
이와 같은 방법을 사용했었다. 하지만 오답이 나왔고, 왜 틀린건지 알 수 없어서 답을 찾아보았다. 결론부터 말하자면 처음 사용한 문제 해결 방법이 틀리지는 않았다. 다만 요일을 표기하는 알파벳이 오타가 있어서 오답 처리되었다. 덕분에 답을 찾게 되면서 배열을 사용하는 더 깔끔한 방법을 찾을 수 있었다. 비교 연산의 수가 줄게 되니 속도도 더 빨라질 것 같다. 앞으로는 정답을 맞추더라도 답을 찾아보아야겠다.

2741

문제

N찍기, 2741번

자연수 N이 주어졌을 때, 1부터 N까지 한 줄에 하나씩 출력하는 프로그램을 작성하시오.

풀이 과정

단순히 for을 통하여 반복하면서 수를 출력하는 문제인데,

로 진행하니까 시간초과가 떴다. endl을 단순 개항문자 \n 으로 바꿔주니 성공했다.

코드

#include <iostream>

using namespace std;

int main(){
    int t;
    cin >> t;
    for(int i = 1; i <= t; i++)
    {
        cout << i << '\n';
    }    
}

소감

말도 안돼는 너무 쉬운 문제라서, 문제의 순서를 가이드 해주는 블로그의 게시글이 잘못 된 줄 알았으나, 그에 보답하듯 보란듯이 문제를 틀려버렸다. 도저히 O(N) 이하의 시간복잡도를 갖도록 구성하는 방법을 모르겠어서 찾아 본 결과, endl은 개행과 동시에 버퍼를 비워주기 때문에 \n을 출력하는것보다 시간이 오래걸린다고 한다. 따라서 실행시간을 줄이기 위해서는 \n을 쓰는게 낫다고 한다.

11721

문제

열 개씩 끊어 출력하기열 개씩 끊어 출력하기, 11721번

알파벳 소문자와 대문자로만 이루어진 길이가 N인 단어가 주어진다.
한 줄에 10글자씩 끊어서 출력하는 프로그램을 작성하시오.

입력 : 첫째 줄에 단어가 주어진다. 단어는 알파벳 소문자와 대문자로만 이루어져 있으며, 길이는 100을 넘지 않는다. 길이가 0인 단어는 주어지지 않는다.

출력 : 입력으로 주어진 단어를 열 개씩 끊어서 한 줄에 하나씩 출력한다. 단어의 길이가 10의 배수가 아닌 경우에는 마지막 줄에는 10개 미만의 글자만 출력할 수도 있다.

풀이 과정

string을 입력받고, 한 글자씩 출력하며 출력할 때 마다 line 변수를 +1, line이 10일때마다 개행을 했다.
string의 크기만큼 for을 돌아야 하는데 C처럼 sizeof(input)/sizeof(char)을 했더니 오류가 발생했다. 따라서 input.size()를 사용했다.

코드

#include <iostream>

using namespace std;

int main()
{
    string input;
    cin >> input;
    int line = 0;
    for (int i = 0; i < input.size(); i++)
    {
        cout << input[i];
        line++;
        if (line%10 == 0)
        {
            cout << endl;
        }
    }
    
}

소감

sizeof 함수를 이용하여 배열의 길이를 알아내려 했는데, string subscript out of range라는 에러가 떴다. 이 에러가 왜 뜨는지에 대해서는 아직 알아내지 못했지만, C++에서는 .size() 함수를 제공하여 다른 방법으로 배열의 길이를 알아낼 수 있었기에 그 방법으로 해결하였다.

11720

문제

숫자의 합, 11720번

N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

입력 : 입력으로 주어진 숫자 N개의 합을 출력한다.

풀이 과정

N을 입력받고, char[] 안에 수를 저장받아 for루프로 합하여 출력한다. 캐릭터 배열은 문자열이니 string 으로 대체하여 풀었다.

코드

#include <iostream>
#include <string>

using namespace std;

int main()
{
    int t, sum = 0;
    cin >> t;
    string input;
    cin >> input;
    
    for (int i = 0; i < t; i++)
    {
        sum = sum + input[i] - '0';
    }
    cout << sum << endl;
    
}

소감

C++의 기본적인 입출력은 이제 조금씩 감이 잡히는 것 같다. 몇문제를 더 풀어보고 나서는 본격적인 알고리즘 풀이로 넘어가야겠다.

그대로 출력하기

문제

그대로 출력하기, 11718번

입력 받은 대로 출력하는 프로그램을 작성하시오.

입력 : 입력이 주어진다. 입력은 최대 100줄로 이루어져 있고, 알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다. 각 줄은 100글자를 넘지 않으며, 빈 줄은 주어지지 않는다. 또, 각 줄은 공백으로 시작하지 않고, 공백으로 끝나지 않는다.

풀이 과정

공백까지 모두 string에 저장하기 위하여 getline함수를 활용하여 한 줄씩 입력받고, cout을 통하여 출력하는 행위를 while 루프를 통해 반복하였다.

코드

#include <iostream>
#include <string>
using namespace std;

int main()
{
    string a;
    
    while(true)
    {
        getline(cin, a);
        if(a == "")
        {   
            break;
        }
        cout << a << endl;
        
    }
}

소감

처음에는 cin으로 이렇게 저렇게 여러 시도를 했으나 문제에서 요구하는 바가 명확히 와닫지 않아서 결국 정답을 찾아보았다. 그 결과 getline 함수를 알게되었고, 이 함수는 한 줄 단위로 string을 받아오는 함수이다. A+B를 벗어나 조금 더 어려운 문제를 풀고 있다는 부분이 실감된다.

A+B-7

문제

A+B - 7, 11021번

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.
입력 : 첫째 줄에 테스트 케이스의 개수 T가 주어진다.
각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)
출력 : 각 테스트 케이스마다 “Case #x: “를 출력한 다음, A+B를 출력한다. 테스트 케이스 번호는 1부터 시작한다.

풀이 과정

for루프와 cin, cout을 이용한 간단한 문제이다. 출력할 때 Case #t: 를 앞에 출력해야 한다는 점만 다를 뿐, 여태까지 했던 문제와 비슷한 유형의 문제였다.

를 이용하여 해결했다.

코드

#include <iostream>

using namespace std;

int main()
{
    int t, a, b;
    cin >> t;
    for (int i = 1; i <= t; i++)
    {
        cin >> a >> b;
        cout << "Case #" << i <<": " << a + b << endl;
    }
   
}

소감

이번 문제는 오히려 전 문제보다 쉬웠다고 생각이 든다. 아직까지는 자신감을 잃지 않고 계속 공부해 나갈만한 것 같다.

A+B-6

문제

A+B - 6, 10953번

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력: 첫째 줄에 테스트 케이스의 개수 T가 주어진다.
각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. A와 B는 콤마(,)로 구분되어 있다. (0 < A, B < 10)

풀이 과정

for문, Cin과 Cout을 이용하여 입출력을 할 때, ‘,’로 구분지어 문자열을 분리하는 것이 관건인 문제이다. string에 입력을 받고, int로 타입캐스팅을 하여 A = string[0], B = string[2]로 계산했는데 값이 틀렸다.
혼자서 해결을 하지 못해서, 찾아본 결과, 아스키코드로 처리되어서 ‘0’을 각각 빼 주면 정확한 값이 된다.

코드

#include <iostream>

using namespace std;

int main()
{
    string input;
    int a, b, t;
    cin >> t;

    for (int i = 0; i < t; i++)
    {
        cin >> input;
        int a = input[0] - '0';
        int b = input[2] - '0';

        cout << a + b << endl;
    }
   
}

소감

1,2를 넣었는데 99가 나와서 도저히 이해하지 못했는데, 아스키 코드를 떠올리지 못한 점이 아쉽다. 간단한 입출력 문제인데도 벌써 막히는 것이 조금 걱정되긴 하지만, 좀 더 열심히 공부해서 이런 문제는 쉽게 넘어갈 수 있도록 해야겠다.

A+B_5

문제

A+B - 5, 10952번

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력 : 입력은 여러 개의 테스트 케이스로 이루어져 있다.
각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)
입력의 마지막에는 0 두 개가 들어온다.

출력 : 각 테스트 케이스마다 A+B를 출력한다

풀이 과정

이 문제는 0이 들어오기 전 까지 계속 입력, 출력을 반복한다. 따라서 while로 무한루프를 만들고, if의 조건으로 0,0이 입력되면 루프를 탈출하는 방법을 사용하였다. 다만 이 방법을 시도하여 1차에는 실패하였다. while 안에서 루프의 종료와 출력의 순서가 틀려서 0+0이 출력되어서 틀렸다.

코드

#include <iostream>
using namespace std;

int main()
{
    int a = 0, b = 0;
    while (1)
    {
        cin >> a >> b;
         if (a == 0 && b == 0)
        {
            break;
        }
        cout << a + b << endl;
    }
}

소감

for과 while, if 까지는 아직 C나 JAVA에서 이용했던 것과 큰 차이가 없어서 어려운 부분을 느끼지는 못했다. 문제 또한 아직 기초에 그쳐서 크게 어렵지는 않았다.

A+B_3

문제

A+B - 3 , 10950번

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력 : 첫째 줄에 테스트 케이스의 개수 T가 주어진다.
각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

출력 : 각 테스트 케이스마다 A+B를 출력한다.

풀이 과정

처음에는 n을 입력받고, for 루프를 통하여 T번의 루트를 실행, 루트 안에서 Cin으로 입력받고, Cout으로 출력할 생각이었으나 오답이었다.

이 때문에 백준에 예시로 나온 것 처럼, for 안에서 a,b를 입력을 다 받은 후에, 출력을 따로 하려고 동적 배열 2개를 할당, 첫 번째 for에서 a와 b를 저장하고 두 번째 for에서 a+b를 출력하였다

코드

#include <iostream>


using namespace std;

int main()
{
    int a = 0, b = 0, c = 1;
    cin >> c;

    int* array_A = new int[c];
    int* array_B = new int[c];

    for (int i = 0; i < c; i++)
    {
        cin >> a >> b;
        array_A[i] = a;
        array_B[i] = b;
    }
    for (int i = 0; i < c; i++)
    {
        cout << array_A[i] + array_B[i] << endl;
    }
}

소감

이전에 풀어본 A+B 문제와 난이도가 너무 확 달라져서 의아해서 정답을 찾아보니 첫 번째 해답이 틀린 이유는 출력시에 개행을 해 주지 않았기 때문이었다. 그래도 덕분에 C++에서 배열을 선언할 변수로 할당하기 위해서는 동적 할당이라는 다른 방법을 써야 한다는 점을 알게 되었던 좋은 접근이었던 것 같다.
