A+B

문제

A + B, 1000번

A와 B를 입력받아서 A+B를 출력하시오

풀이 과정

아직 알고리즘에 대하여 생각할 부분은 전혀 없기 때문에, C++ 문법을 익히는데 중점을 두었고, using namespace를 통하여 std를 매번 선언하지 않아도 되도록 하는 방법을 사용하였고, cout과 cin을 이용하여 입출력을 처리하였다

코드

#include <iostream>

int main()
{
    using namespace std;
    
    int x = 0, y = 0;
    cin >> x;
    cin >> y;
    
    cout << x+y;

    return 0;
}

소감

cout과 cin이란 함수 이름이 생각보다 직관성이 있어서 printf와 scanf에만 익숙해 져 있었는데 생각보다 쉽게 적응 할 수 있었던 것 같다. using namespace는 상당히 유용한 기능인 것 같으니 꼭 숙지해 두어야겠다.

Hello World

문제

Hello World, 2557번

Hello World를 출력하시오

풀이 과정

C++ 로 알고리즘 공부를 진행할 예정이고, 아주 간단한 출력문이라 알고리즘에 대한 분석 보다는 C++ 문법에 대한 이해를 더 중요시 하였다.

C++에 대한 지식이 전혀 없어서 검색을 통하여 출력 문법을 알아보았다.

코드

#include <iostream>

int main()
{
    std::cout << "Hello World!";
    
    return 0;
}

소감

처음 접하는 C++이고, C랑은 기초 문법을 공유한다고 들었는데, 간단한 출력부터 생각보다 많이 다른 모습에 당황했지만, 아직은 큰 난관을 겪지 못했고, 계속 열심히 진행 할 생각이다.
