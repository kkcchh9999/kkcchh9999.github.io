별 찍기-12

문제

별 찍기-12, 2522번

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.
입력 : 3

풀이 과정

이 문제 또한 다른 별찍기와 크게 다르지 않지만, 이 문제를 풀 때는 위와 아래로 분리하지 않고, 하나의 for 안에서 윗부분과 아랫부분을 모두 출력했다.
입력이 n이고 for 루프의 i가 1 ~ 2n-1 일때, 각 줄에는 |n-i|개의 공백이 출력되고, n- |n-i|개의 별이 출력된다. 이를 이용해 해결했다.

코드

#include <iostream>

using namespace std;

int main() {

    int n;
    cin >> n;

    for (int i = 1; i <= n * 2 - 1; i++)
    {
        for (int j = 0; j < abs(n - i); j++)
            cout << " ";
        for (int j = 0; j < n - abs(n - i); j++)
            cout << "*";

        cout << endl;
    }
}

소감

이 문제의 해답은 처음에 위 아래를 분리하여 풀고 난 후에, 어떻게하면 한번에 위 아래를 다 출력할 수 있을지 답을 찾아봐서 알게 된 경우이다. 이 경우에 코드가 더 깔끔해지는 것 같고, 절대값을 표기하는 abs라는 함수도 알게되었다. 경우에 따라 절대값도 활용할 수 있도록 시야를 넓혀 갈 필요가 있겠다.

이 문제 이후에도 별 찍기 문제를 몇문제 더 풀었지만, 난이도도 그렇게 높지 않고, 새로운 C++ 함수를 알게 된 문제도 아니라서 포스팅은 하지 않고, 여기까지 입출력 문제에 해당하는 문제는 모두 풀었다. 다음 포스팅부터는 DP에 해당하는 문제이다.

별 찍기-8

문제

별 찍기-8, 2445번

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.
입력 : 5
출력 :

풀이 과정

위쪽의 5줄, 즉 n줄과 아래쪽의 4줄, 즉 n-1줄을 분리하여 분석, 출력하였다.
위쪽 :
첫째줄 : 1개의 별, 2(n-1)개의 공백, 1개의 별
둘째줄 : 2개의 별, 2(n-1)개의 공백, 2개의 별
⁝
n째줄 : n개의 별, 0개의 공백, n 개의 별을 출력한다.
아래쪽 :
첫째줄 : n-1개의 별, 2개의 공백, n-1개의 별
둘째줄 : n-2개의 별, 4개의 공백, n-1개의 별
⁝
n-1쨰줄 : n-(n-1)개의 별, 2(n-1)개의 공백, n-(n-1)개의 별

코드

#include <iostream>

using namespace std;

int main() {

    int n;
    cin >> n;

    for (int i = 1; i <= n; i++)
    {
        for (int j = 0; j < i; j++)
        {
            cout << "*";
        }

        for (int k = i*2; k < 2 * n  ; k++)
        {
            cout << " ";
        }
        for (int j = 0; j < i; j++)
        {
            cout << "*";
        }
        
        cout << endl;
    }
    for (int i = 1; i < n; i++)
    {
        for (int j = i; j < n; j++)
        {
            cout << "*";
        }
        for (int j = 0; j < i * 2; j++)
        {
            cout << " ";
        }
        for (int j = i; j < n; j++)
        {
            cout << "*";
        }

        cout << endl;
    }

}

소감

별 찍기 예제를 몇가지 더 풀어 보았지만, 그 중에서 그래도 나름 난이도가 있는 문제를 추렸다. C언어를 처음 배울 때 했던 예제와 비슷해서 떠올리는데 큰 어려움은 없었다.

최소, 최대

문제

최소, 최대, 10818번

N개의 정수가 주어진다. 이때, 최솟값과 최댓값을 구하는 프로그램을 작성하시오.

입력 : 첫째 줄에 정수의 개수 N (1 ≤ N ≤ 1,000,000)이 주어진다. 둘째 줄에는 N개의 정수를 공백으로 구분해서 주어진다. 모든 정수는 -1,000,000보다 크거나 같고, 1,000,000보다 작거나 같은 정수이다.

풀이 과정

최소값인 -1000000을 max, 최대값인 1000000을 min으로 두고, n번의 루프동안 cin으로 입력받는 수를 max와 min 각각 한번씩 비교하여, max보다 크면 max로 저장, min보다 작으면 min으로 저장하여 출력한다.

코드

#include <iostream>

using namespace std;

int main() {

    int n;
    int max = -1000000, min = 1000000;
    int tmp;

    cin >> n;
    
    for (int i = 0; i < n; i++)
    {
        cin >> tmp;
        if (max < tmp)
        {
            max = tmp;
        }
        if (min > tmp)
        {
            min = tmp;
        }
    }
    cout << min << " " << max << endl;

}

소감

최대 1000000개의 입력이 들어오기 때문에, 모든 입력을 다 받은 후에 비교하는것은 불가능해보였고, 입력이 들어올때마다 비교하여 저장하고 출력하는 방식을 선택했다. 이제 슬슬 1학년때 처음 C언어를 배우던 때의 과제 정도의 수준을 갖는 문제가 나오고 있는 것 같다. 입력값의 범위를 잘못 보아 한번 틀렸는데, 문제를 잘 읽도록 해야겠다.

합

문제

합, 8393번

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

풀이 과정

단순히 for루프를 이용하여 1n까지 더하면 O(n)이 된다. 하지만 약간의 수학을 이용하여 1n 까지의 합은 n(n+1) / 2 인 점을 이용하면 O(1)로 만들수 있다.

코드

#include <iostream>

using namespace std;

int main() {

    int n;
    cin >> n;
    cout << (n * (n + 1) / 2);
}

소감

이제 C++언어가 조금씩은 익숙해 진 것 같다. 이제 조금씩 더 좋은 알고리즘을 생각하기 위해 노력할 만한 문제들이 나오지 않을까 기대가 된다.

2007년

문제

2007년, 1924번

오늘은 2007년 1월 1일 월요일이다. 그렇다면 2007년 x월 y일은 무슨 요일일까? 이를 알아내는 프로그램을 작성하시오.

입력 : 첫째 줄에 빈 칸을 사이에 두고 x(1 ≤ x ≤ 12)와 y(1 ≤ y ≤ 31)이 주어진다. 참고로 2007년에는 1, 3, 5, 7, 8, 10, 12월은 31일까지, 4, 6, 9, 11월은 30일까지, 2월은 28일까지 있다.

출력 : 첫째 줄에 x월 y일이 무슨 요일인지에 따라 SUN, MON, TUE, WED, THU, FRI, SAT중 하나를 출력한다.

풀이 과정

일주일은 7일이고, 이를 이용해 나머지 연산을 통하여 문제를 해결한다. x월일 경우, (1월의 일 수(31) + 2월의 일 수(28)+….(x-1)월의 일 수(??) + y )을 하면 총 몇일인지 변환이 가능하고, 여기에 % 7 연산을 한 결과에 따라 요일을 출력하는 방식으로 해결한다.

코드

#include

using namespace std;

int main() {

int x, y, tmp = 0;
cin >> x >> y;

for (int i = 1; i < x; i++)
{
    if (i == 1 || i == 3 || i == 5 || i == 7 || i == 8 || i == 10 || i == 12)
        tmp = tmp + 31;
    else if (i == 2)
        tmp = tmp + 28;
    else
        tmp = tmp + 30;
}
tmp = tmp + y;
if (tmp % 7 == 1)
    cout << "MON" << endl;
else if (tmp % 7 == 2)
    cout << "TUE" << endl;
else if (tmp % 7 == 3)
    cout << "WED" << endl;
else if (tmp % 7 == 4)
    cout << "THR" << endl;
else if (tmp % 7 == 5)
    cout << "FRI" << endl;
else if (tmp % 7 == 6)
    cout << "SAT" << endl;
else
    cout << "SON" << endl;

}

```
이와 같은 방법을 사용했었다. 하지만 오답이 나왔고, 왜 틀린건지 알 수 없어서 답을 찾아보았다. 결론부터 말하자면 처음 사용한 문제 해결 방법이 틀리지는 않았다. 다만 요일을 표기하는 알파벳이 오타가 있어서 오답 처리되었다. 덕분에 답을 찾게 되면서 배열을 사용하는 더 깔끔한 방법을 찾을 수 있었다. 비교 연산의 수가 줄게 되니 속도도 더 빨라질 것 같다. 앞으로는 정답을 맞추더라도 답을 찾아보아야겠다.

N찍기

문제

N찍기, 2741번

자연수 N이 주어졌을 때, 1부터 N까지 한 줄에 하나씩 출력하는 프로그램을 작성하시오.

풀이 과정

단순히 for을 통하여 반복하면서 수를 출력하는 문제인데,

로 진행하니까 시간초과가 떴다. endl을 단순 개항문자 \n 으로 바꿔주니 성공했다.

코드

#include <iostream>

using namespace std;

int main(){
    int t;
    cin >> t;
    for(int i = 1; i <= t; i++)
    {
        cout << i << '\n';
    }    
}

소감

말도 안돼는 너무 쉬운 문제라서, 문제의 순서를 가이드 해주는 블로그의 게시글이 잘못 된 줄 알았으나, 그에 보답하듯 보란듯이 문제를 틀려버렸다. 도저히 O(N) 이하의 시간복잡도를 갖도록 구성하는 방법을 모르겠어서 찾아 본 결과, endl은 개행과 동시에 버퍼를 비워주기 때문에 \n을 출력하는것보다 시간이 오래걸린다고 한다. 따라서 실행시간을 줄이기 위해서는 \n을 쓰는게 낫다고 한다.

열 개씩 끊어 출력하기

문제

열 개씩 끊어 출력하기, 11721번

알파벳 소문자와 대문자로만 이루어진 길이가 N인 단어가 주어진다.
한 줄에 10글자씩 끊어서 출력하는 프로그램을 작성하시오.

입력 : 첫째 줄에 단어가 주어진다. 단어는 알파벳 소문자와 대문자로만 이루어져 있으며, 길이는 100을 넘지 않는다. 길이가 0인 단어는 주어지지 않는다.

출력 : 입력으로 주어진 단어를 열 개씩 끊어서 한 줄에 하나씩 출력한다. 단어의 길이가 10의 배수가 아닌 경우에는 마지막 줄에는 10개 미만의 글자만 출력할 수도 있다.

풀이 과정

string을 입력받고, 한 글자씩 출력하며 출력할 때 마다 line 변수를 +1, line이 10일때마다 개행을 했다.
string의 크기만큼 for을 돌아야 하는데 C처럼 sizeof(input)/sizeof(char)을 했더니 오류가 발생했다. 따라서 input.size()를 사용했다.

코드

#include <iostream>

using namespace std;

int main()
{
    string input;
    cin >> input;
    int line = 0;
    for (int i = 0; i < input.size(); i++)
    {
        cout << input[i];
        line++;
        if (line%10 == 0)
        {
            cout << endl;
        }
    }
    
}

소감

sizeof 함수를 이용하여 배열의 길이를 알아내려 했는데, string subscript out of range라는 에러가 떴다. 이 에러가 왜 뜨는지에 대해서는 아직 알아내지 못했지만, C++에서는 .size() 함수를 제공하여 다른 방법으로 배열의 길이를 알아낼 수 있었기에 그 방법으로 해결하였다.

숫자의 합

문제

숫자의 합, 11720번

N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

입력 : 입력으로 주어진 숫자 N개의 합을 출력한다.

풀이 과정

N을 입력받고, char[] 안에 수를 저장받아 for루프로 합하여 출력한다. 캐릭터 배열은 문자열이니 string 으로 대체하여 풀었다.

코드

#include <iostream>
#include <string>

using namespace std;

int main()
{
    int t, sum = 0;
    cin >> t;
    string input;
    cin >> input;
    
    for (int i = 0; i < t; i++)
    {
        sum = sum + input[i] - '0';
    }
    cout << sum << endl;
    
}

소감

C++의 기본적인 입출력은 이제 조금씩 감이 잡히는 것 같다. 몇문제를 더 풀어보고 나서는 본격적인 알고리즘 풀이로 넘어가야겠다.

그대로 출력하기

문제

그대로 출력하기, 11718번

입력 받은 대로 출력하는 프로그램을 작성하시오.

입력 : 입력이 주어진다. 입력은 최대 100줄로 이루어져 있고, 알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다. 각 줄은 100글자를 넘지 않으며, 빈 줄은 주어지지 않는다. 또, 각 줄은 공백으로 시작하지 않고, 공백으로 끝나지 않는다.

풀이 과정

공백까지 모두 string에 저장하기 위하여 getline함수를 활용하여 한 줄씩 입력받고, cout을 통하여 출력하는 행위를 while 루프를 통해 반복하였다.

코드

#include <iostream>
#include <string>
using namespace std;

int main()
{
    string a;
    
    while(true)
    {
        getline(cin, a);
        if(a == "")
        {   
            break;
        }
        cout << a << endl;
        
    }
}

소감

처음에는 cin으로 이렇게 저렇게 여러 시도를 했으나 문제에서 요구하는 바가 명확히 와닫지 않아서 결국 정답을 찾아보았다. 그 결과 getline 함수를 알게되었고, 이 함수는 한 줄 단위로 string을 받아오는 함수이다. A+B를 벗어나 조금 더 어려운 문제를 풀고 있다는 부분이 실감된다.
