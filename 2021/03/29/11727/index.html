    2 x n 타일링

    작성일 2021-03-29 Edited on 2021-03-30 In 알고리즘

    문제

    2 x n 타일링 2, 11727번

    2×n 직사각형을 1×2, 2×1과 2×2 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.
    아래 그림은 2×17 직사각형을 채운 한가지 예이다.

    입력 : 첫째 줄에 n이 주어진다. 출력 : 둘째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.

    풀이 과정

    이전에 풀었던 타일링과 같은 방법이지만, DP[n-2]에서 1x2 두장을 더하는 방법과, 2x2 한장을 더하는 방법까지 두 가지의 방법이 생겼기 때문에, DP[n] = DP[n-1] + (DP[n-2] x 2)를 이용하여 풀 수 있다.

    코드

    #include <iostream>

using namespace std;

int main() {

    int DP[1001] = { 0, };
    int n = 0, answer = 0;

    cin >> n; 
    DP[1] = 1;
    DP[2] = 3;
    for (int i = 3; i <= n; i++)
    {
        DP[i] = (DP[i - 1] + (DP[i - 2] * 2))%10007;
    }
    cout << DP[n] << endl;
}
