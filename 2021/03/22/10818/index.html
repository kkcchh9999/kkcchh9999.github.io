최소, 최대

문제

최소, 최대, 10818번

N개의 정수가 주어진다. 이때, 최솟값과 최댓값을 구하는 프로그램을 작성하시오.

입력 : 첫째 줄에 정수의 개수 N (1 ≤ N ≤ 1,000,000)이 주어진다. 둘째 줄에는 N개의 정수를 공백으로 구분해서 주어진다. 모든 정수는 -1,000,000보다 크거나 같고, 1,000,000보다 작거나 같은 정수이다.

풀이 과정

최소값인 -1000000을 max, 최대값인 1000000을 min으로 두고, n번의 루프동안 cin으로 입력받는 수를 max와 min 각각 한번씩 비교하여, max보다 크면 max로 저장, min보다 작으면 min으로 저장하여 출력한다.

코드

#include <iostream>

using namespace std;

int main() {

    int n;
    int max = -1000000, min = 1000000;
    int tmp;

    cin >> n;
    
    for (int i = 0; i < n; i++)
    {
        cin >> tmp;
        if (max < tmp)
        {
            max = tmp;
        }
        if (min > tmp)
        {
            min = tmp;
        }
    }
    cout << min << " " << max << endl;

}

소감

최대 1000000개의 입력이 들어오기 때문에, 모든 입력을 다 받은 후에 비교하는것은 불가능해보였고, 입력이 들어올때마다 비교하여 저장하고 출력하는 방식을 선택했다. 이제 슬슬 1학년때 처음 C언어를 배우던 때의 과제 정도의 수준을 갖는 문제가 나오고 있는 것 같다. 입력값의 범위를 잘못 보아 한번 틀렸는데, 문제를 잘 읽도록 해야겠다.
