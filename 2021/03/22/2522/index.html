별 찍기-12

문제

별 찍기-12, 2522번

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.
입력 : 3

풀이 과정

이 문제 또한 다른 별찍기와 크게 다르지 않지만, 이 문제를 풀 때는 위와 아래로 분리하지 않고, 하나의 for 안에서 윗부분과 아랫부분을 모두 출력했다.
입력이 n이고 for 루프의 i가 1 ~ 2n-1 일때, 각 줄에는 |n-i|개의 공백이 출력되고, n- |n-i|개의 별이 출력된다. 이를 이용해 해결했다.

코드

#include <iostream>

using namespace std;

int main() {

    int n;
    cin >> n;

    for (int i = 1; i <= n * 2 - 1; i++)
    {
        for (int j = 0; j < abs(n - i); j++)
            cout << " ";
        for (int j = 0; j < n - abs(n - i); j++)
            cout << "*";

        cout << endl;
    }
}

소감

이 문제의 해답은 처음에 위 아래를 분리하여 풀고 난 후에, 어떻게하면 한번에 위 아래를 다 출력할 수 있을지 답을 찾아봐서 알게 된 경우이다. 이 경우에 코드가 더 깔끔해지는 것 같고, 절대값을 표기하는 abs라는 함수도 알게되었다. 경우에 따라 절대값도 활용할 수 있도록 시야를 넓혀 갈 필요가 있겠다.

이 문제 이후에도 별 찍기 문제를 몇문제 더 풀었지만, 난이도도 그렇게 높지 않고, 새로운 C++ 함수를 알게 된 문제도 아니라서 포스팅은 하지 않고, 여기까지 입출력 문제에 해당하는 문제는 모두 풀었다. 다음 포스팅부터는 DP에 해당하는 문제이다.
