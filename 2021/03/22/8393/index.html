합

문제

합, 8393번

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

풀이 과정

단순히 for루프를 이용하여 1n까지 더하면 O(n)이 된다. 하지만 약간의 수학을 이용하여 1n 까지의 합은 n(n+1) / 2 인 점을 이용하면 O(1)로 만들수 있다.

코드

#include <iostream>

using namespace std;

int main() {

    int n;
    cin >> n;
    cout << (n * (n + 1) / 2);
}

소감

이제 C++언어가 조금씩은 익숙해 진 것 같다. 이제 조금씩 더 좋은 알고리즘을 생각하기 위해 노력할 만한 문제들이 나오지 않을까 기대가 된다.
