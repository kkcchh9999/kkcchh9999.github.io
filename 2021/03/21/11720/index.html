    숫자의 합

    작성일 2021-03-22 Edited on 2021-03-23 In 알고리즘

    문제

    숫자의 합, 11720번

    N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

    입력 : 입력으로 주어진 숫자 N개의 합을 출력한다.

    풀이 과정

    N을 입력받고, char[] 안에 수를 저장받아 for루프로 합하여 출력한다. 캐릭터 배열은 문자열이니 string 으로 대체하여 풀었다.

    코드

    #include <iostream>
#include <string>

using namespace std;

int main()
{
    int t, sum = 0;
    cin >> t;
    string input;
    cin >> input;
    
    for (int i = 0; i < t; i++)
    {
        sum = sum + input[i] - '0';
    }
    cout << sum << endl;
    
}
