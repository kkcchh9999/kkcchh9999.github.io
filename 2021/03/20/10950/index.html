    A+B-3

    작성일 2021-03-20 Edited on 2021-03-23 In 알고리즘

    문제

    A+B - 3 , 10950번

    두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

    입력 : 첫째 줄에 테스트 케이스의 개수 T가 주어진다.
    각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

    출력 : 각 테스트 케이스마다 A+B를 출력한다.

    풀이 과정

    처음에는 n을 입력받고, for 루프를 통하여 T번의 루트를 실행, 루트 안에서 Cin으로 입력받고, Cout으로 출력할 생각이었으나 오답이었다.

    이 때문에 백준에 예시로 나온 것 처럼, for 안에서 a,b를 입력을 다 받은 후에, 출력을 따로 하려고 동적 배열 2개를 할당, 첫 번째 for에서 a와 b를 저장하고 두 번째 for에서 a+b를 출력하였다

    코드

    #include <iostream>


using namespace std;

int main()
{
    int a = 0, b = 0, c = 1;
    cin >> c;

    int* array_A = new int[c];
    int* array_B = new int[c];

    for (int i = 0; i < c; i++)
    {
        cin >> a >> b;
        array_A[i] = a;
        array_B[i] = b;
    }
    for (int i = 0; i < c; i++)
    {
        cout << array_A[i] + array_B[i] << endl;
    }
}

    소감

    이전에 풀어본 A+B 문제와 난이도가 너무 확 달라져서 의아해서 정답을 찾아보니 첫 번째 해답이 틀린 이유는 출력시에 개행을 해 주지 않았기 때문이었다. 그래도 덕분에 C++에서 배열을 선언할 변수로 할당하기 위해서는 동적 할당이라는 다른 방법을 써야 한다는 점을 알게 되었던 좋은 접근이었던 것 같다.
